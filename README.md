# Curriculum
**Week 1: Transformer Reverse Engineering Foundations**

- **Topics:** Residual Stream, Attention Heads (Q-K/O-V Circuits), Logit Lens
- **Main Paper:** *A Mathematical Framework for Transformer Circuits* (Elhage et al., 2021)
- **Blog/Resource:** interpreting gpt: the logit lens (nostalgebraist) & TransformerLens Main Tutorial

**Week 2: Physical Reality of Memory & Knowledge Editing**

- **Topics:** FFN as Key-Value Memory, Causal Tracing, Factual Association
- **Main Papers:**
- 
    - *Transformer Feed-Forward Layers Are Key-Value Memories* (Geva et al., 2020)
    - *Locating and Editing Factual Associations in GPT (ROME)* (Meng et al., 2022)
- **Blog/Resource:** ROME Interactive Demo Website (rome.baulab.info)

**Week 3: In-Context Learning & Circuit Discovery**

- **Topics:** Induction Heads, Path Patching, Automated Circuit Discovery (ACDC)
- **Main Papers:**
- 
    - *In-context Learning and Induction Heads* (Olsson et al., 2022)
    - *Towards Automated Circuit Discovery for Mechanistic Interpretability* (Conmy et al., 2023)
- **Blog/Resource:** Anthropic's Transformer Circuits Thread on Induction Heads

**Week 4: Dynamics of Generalization & Geometric Grokking**

- **Topics:** Phase Transitions, Representational Magnitude, Fully Bounded Spherical Topology, Uniform Attention Ablation
- **Main Paper:**
- 
    - *The Geometric Inductive Bias of Grokking: Bypassing Phase Transitions via Architectural Topology* (Yıldırım, 2026)
- **Blog/Resource:** A Mechanistic Interpretability Analysis of Grokking (Neel Nanda's Blog) / Avala.ai: Grokking - How AI Learns

**Week 5: Resolving Polysemanticity with Temporal SAEs**

- **Topics:** Polysemanticity, Sparse Autoencoders (SAE), Temporal Contrastive Loss, Semantic vs. Syntactic Disentanglement
- **Main Paper:**
- 
    - *Temporal Sparse Autoencoders: Leveraging the Sequential Nature of Language for Interpretability* (Bhalla et al., 2025/2026)
- **Blog/Resource:** DeepMind Safety Research: Negative Results for Sparse Autoencoders on Downstream Tasks

**Week 6: Mechanistic Continual Learning & Factual Recall**

- **Topics:** Gradient Interference, Representation Drift, Catastrophic Forgetting, Lookup-Table Mechanism
- **Main Papers:**
- 
    - *Mechanistic Analysis of Catastrophic Forgetting in Large Language Models During Continual Fine-tuning* (Laitinen Imanov, 2026)
    - *Robust Knowledge Unlearning and Editing via Mechanistic Localization* (Guo et al., ICLR 2025)
- **Blog/Resource:** GitHub: olaflaitinen / Mechanistic Analysis of Catastrophic Forgetting Code (or similar continual learning repos)
