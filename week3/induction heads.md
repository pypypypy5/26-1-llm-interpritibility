# 정의
[[transformer]]에서 두 개의 연속된 head가 이전의 패턴을 참고하여 다음에 어떤 토큰이 올지를 추측한다.

...(A)(B)...A* (?) 와 같은 구조라 하자.
이제 A* 뒤에 나올 ? 토큰을 맞춰야 한다.

첫번째 head에서 앞의 토큰들 중 A* 와 유사한 A 뒤에 B가 나왔다는 것을 인식하고 B를 attention, 이 정보를 토큰에 압축시킨 후 뒤로 보내면, 
다음 head는 받은 정보를 바탕으로 앞의 B를 참고하여 유사한 B* 를 결과 토큰으로 뒤로 보낸다.

### 작동 방식
① 이전 토큰 헤드 (Previous Token Head) - 주로 레이어 $L$에 위치

- **역할:** 현재 위치에 '바로 직전 토큰이 무엇이었는지'에 대한 정보를 복사
- **작동:** 어텐션 가중치가 항상 현재 위치($i$)에서 바로 이전 위치($i-1$)를 가리키도록 학습
- **결과:** 이 헤드를 거치면, 특정 위치의 잔차 연결(Residual Stream)에는 "내 앞에 [A]가 있었다"라는 벡터 정보가 주입

② 유도 헤드 (Induction Head) - 주로 레이어 $L+1$ 이상에 위치
- **역할:** 이전 토큰 헤드가 심어놓은 정보를 바탕으로 패턴을 완성
- 가중치 결합: 유도 헤드의 **$W_Q$는 현재 토큰을 바라보고, $W_K$는 이전 레이어에서 '이전 토큰 정보'가 복사된 위치를 찾는다

### **Fuzzy Induction** 
실제 가중치 행렬은 완전 일치(`A=A`)뿐만 아니라 **유사한 의미**를 가진 벡터들끼리도 반응하도록 학습됨. 
ex. "프랑스 - 파리"라는 관계를 보고, 나중에 "이탈리아"가 나오면 "로마"를 떠올리는 식의 '추상적 유도'가 가능

# [[ICL]]의 원인이라는 근거
- **거시적 동시 발생 (Macroscopic co-occurrence):** 언어 모델을 훈련하는 초기 단계에서 특정한 '위상 변화(Phase Change, 급격한 변화)'가 관찰됩니다. 놀랍게도 모델 내부에 **귀납 헤드가 형성되는 정확히 그 시점**에 모델의 문맥 내 학습 능력이 폭발적으로 향상됩니다.
    
- **거시적 동시 섭동 (Macroscopic co-perturbation):** 연구진이 모델의 아키텍처를 의도적으로 변형하여 귀납 헤드가 형성되는 시기를 늦추거나 앞당겼을 때, 문맥 내 학습 능력이 급증하는 시기 역시 그에 맞춰 정확히 똑같이 이동했습니다.
    
- **직접적 절제 (Direct ablation):** 소형 모델을 대상으로 실험할 때, 추론 단계에서 귀납 헤드를 인위적으로 비활성화(제거)해버리면 모델의 문맥 내 학습 능력이 크게 떨어지는 것을 확인했습니다.
    
- **기능의 일반성 (Examples of generality):** 귀납 헤드는 단순히 `[A][B]`를 똑같이 복사하는 것을 넘어, 비슷한 패턴이나 번역, 추상적인 규칙 매칭 같은 매우 다양하고 복잡한 문맥 내 학습 동작까지 수행할 수 있음이 관찰되었습니다.
    
- **기계적 타당성 (Mechanistic plausibility):** 소형 모델을 기계적 해석학(Mechanistic Interpretability) 관점에서 분석한 결과, 귀납 헤드의 작동 구조 자체가 단순한 패턴 복사를 넘어 일반적인 문맥 학습 용도로 재사용되기에 매우 적합한 형태를 띠고 있습니다.
    
- **소형 모델에서 대형 모델로의 연속성 (Continuity):** 강력한 인과관계 증명은 주로 소형 모델에서 이루어졌지만, 귀납 헤드의 형성 양상이나 문맥 내 학습 능력 간의 상관관계 데이터가 대규모 모델에 이르기까지 부드럽고 연속적으로 나타납니다. 이는 대형 모델에서도 동일한 메커니즘이 작동한다는 것을 강하게 시사합니다.
